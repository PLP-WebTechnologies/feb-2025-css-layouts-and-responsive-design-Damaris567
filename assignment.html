Trying out the use of flex boxes and CSS Grids. Flexbox and CSS Grid are two powerful layout systems in CSS used to create responsive and flexible web designs. Flexbox (Flexible Box Layout) Flexbox is a one-dimensional layout system designed to arrange items in a row or column. It is ideal for distributing space and aligning items within a container. Key Features: Works in one direction at a time (row or column). Aligns items vertically
